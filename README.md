# 🐬 Falphin – AI Productivity Companion

Falphin is a gamified productivity planner that helps users organize tasks,
prioritize deadlines, and stay motivated through a virtual companion that
evolves as tasks are completed.

## Features

- Task management system
- AI-based task prioritization
- Gamified Falphin evolution
- Productivity dashboard
- Task priority analytics

## Tech Stack

- Python
- Streamlit
- Matplotlib

## How to Run

Clone the repository:

git clone https://github.com/Akira2006/falphin-productivity-planner.git

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## Demo

The application allows users to add tasks, receive AI task prioritization,
track productivity, and evolve the Falphin companion based on progress.
